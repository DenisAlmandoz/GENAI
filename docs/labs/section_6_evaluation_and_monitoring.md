# Section 6 Lab - Evaluation and Monitoring

## What you will practice
- Model selection using evaluation metrics.
- RAG evaluation with MLflow-compatible thinking.
- Monitoring live endpoints with inference logging.
- Cost control and lifecycle comparisons.

## Exercises
1. Define offline evaluation metrics for retrieval, generation quality, latency, and safety.
2. Use `monitoring_metrics()` to build a live dashboard spec.
3. Explain which evaluation judges require ground truth and which do not.
4. Compare evaluation before deployment vs. monitoring after deployment.
5. Describe how to use inference tables and Agent Monitoring for continuous improvement.

## Check yourself
- Did you connect metrics to user and business outcomes?
- Did you include both offline and online measurements?
- Did you distinguish regression testing from production observability?
